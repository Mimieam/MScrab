MScrab
======

The live version is [here](http://echimane.com/scrabb/)
#####Basic Squeleton
<pre>
World
	Player -> currentScore
     	   -> tileHolder
           -> timer
           
 	Board 
    	   -> Grid 
           		-> placeHolder
           -> Coordinate ruler (only 4 debugging)		
                
    Tile
    	   -> Type
           -> Coodinates
           -> value
           -> direction
                      
    Other -> Menu
    	  -> Minimap
          -> debug Menu
             
</pre>
    
#####Tile types:
* num
* sign (operator & <del>parenteses</del>) 
* equal
	-> keep 2 running totals , vertical/horizontal 


#####placeHolder types:
* single - num
* double - num
* triple - eq
